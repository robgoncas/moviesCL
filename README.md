# Catalog App

A modern **Vue 3** project for showcasing a **product catalog** with individual product detail pages. Built with **TypeScript** and **Vuetify 3 (Material 3)** for a clean, dynamic, and responsive UI.  

This project serves as a foundation for small e-commerce apps, portfolios, or product showcases.
